# Flowset

[Flowset](https://flowset.io/) is an open source business process automation platform for passionate software engineers.

### Platform Components

- **[Flowset Control](https://github.com/flowset/flowset-control-community)**. Web application to maintain and support the process applications in a specialized admin environment without any workflow interruptions.
- **[Flowset Studio](https://plugins.jetbrains.com/plugin/25655-openbpm/)**. Plugin for IntelliJ IDEA to develop enterprise-grade process applications
- **[Flowset Tasklist](https://github.com/flowset/flowset-tasklist-react-community)**. Web application for enabling end-users to work with tasks.
